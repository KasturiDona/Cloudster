Description

HP LiveVault from Cloudster is a fully managed cloud data protection service for SMB’s that automates, streamlines, and reduces complexity of server data protection. Through the LiveVault Web Management Portal, organisations can manage and monitor the entire data protection workflow including backup, offsite storage, and recovery anytime, anywhere. With 96 recovery points daily, HP LiveVault enables organisations to better achieve their RTOs and RPOs. The highly cost effective SMB plans comes with simple deployment and management.

Install the client software on the target servers, decide on your retention time, choose the data sets you would like to back up –and then relax. The client software will automatically backup the server using a schedule you define, or in real-time. You can choose how long you would like to store sets of data, from one month to seven years. You can even select to store the data onsite –as well as in the cloud.

Continuous Data Protection (CDP) –backup in real-time without affecting network traffic and improve your Recovery Point Objectives (RPO) with up to minute data recovery and choose a retention period that suits you, up to seven years.

HP seamlessly integrates with all popular Exchange and SQL-based programs.
