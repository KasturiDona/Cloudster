Virtual Data Centre (CVDC)

Overview

  • Service: Virtual Data Centre, based on VMware’s vCloud
  • Function: Virtual Servers
  • Commercial model: Pay-as-you go service with no upfront costs

Description

Cloudster has simplified the setup and management of a single or multiple servers in the Cloud to the point where almost no IT skills are required. Using the Cloudster control panel to configure the resources the environment is created in real time. Then configure your servers and networking using the VMware vCloud director.

The key benefits of Cloudster VMware vCloud:

  • Intuitive and simple provisioning that is almost instant, enabling infrastructure to be ready for use in minutes.
  • Cloudster’s vCloud removes the need for hardware and reduces management costs by making it possible to virtualise and outsource selected applications directly to Cloudster’s reliable, secure, cost-effective cloud-based server solution.
  • Cloudster’s vCloud comes with a 99.9% uptime guarantee.
  • Reliable, 24/7 Data Centre Connectivity connectivity between the users and the vCloud environment over the Internet connection and/or Site-to-site VPN.
  • Build internal and external connections between local and remote applications utilising centralised management.
